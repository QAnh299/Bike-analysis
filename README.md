# 🚲 Bike Analysis

## 📖 Giới thiệu
Đồ án môn học **Kho dữ liệu và Kinh doanh thông minh**.

### 📌 Đề tài
**Xây dựng kho dữ liệu hỗ trợ báo cáo tình hình bán hàng của Evans Cycles**

Dự án tập trung vào việc:
- Xây dựng kho dữ liệu (Data Warehouse)
- Thực hiện quy trình ETL
- Phân tích và trực quan hóa dữ liệu bán hàng
- Hỗ trợ báo cáo và ra quyết định kinh doanh

## 🛠 Công nghệ sử dụng

- **SQL Server** — Lưu trữ và quản lý dữ liệu
- **SSIS (SQL Server Integration Services)** — Thực hiện ETL dữ liệu
- **Power BI** — Trực quan hóa và xây dựng báo cáo

## 📊 Nội dung thực hiện

- Thiết kế mô hình kho dữ liệu

- Xây dựng dashboard phân tích bán hàng
- Báo cáo doanh thu bán hàng, khách hàng và sản phẩm
